+======================================+
   Discuz! X3 Beta 20130305
+======================================+
新增 移動瀏覽全方位適配。
新增 面向非智能機的低端wap版，面向iphone、安卓高端機的高端觸摸版
新增 獨立移動客戶端（iphone/android）支持。客戶端API功能更加豐富，一鍵生成屬於自己論壇的獨立客戶端
新增 門戶資訊頁面靜態化，同時支持URL別名自定義。在增強服務器負載的同時，優化站點的SEO能力。
新增 論壇支持https、IPV6安全鏈接訪問 ，讓特殊環境下的論壇數據吞吐更加安全
新增 版塊的收藏人數、發帖數排行、以及每日版塊發帖數排名變化，增加版主攀比動力，讓版塊活躍起來
新增 遊客查看主題的時候不再顯示額外用戶數據，大幅降低數據庫查詢數
新增 論壇首頁四格標配，讓論壇的熱帖、新帖等可以更好的展示給用戶
新增 優化大師，全面檢測論壇的安全性和性能優化設置，引導站長對論壇進行更好的優化調整
新增 針對指定版塊的主題、資訊、日誌進行防採集設置，讓採集本站內容不再那麼容易
新增 對帖子查看頁面用戶資料信息根據用戶組進行單獨設置，讓不同的用戶組，顯示的資料不一樣，更顯獨特身份
新增 控制註冊新用戶的時候，是否郵箱必填
新增 可以對版塊回復框進行背景圖片設置。這可以更有利於版規需知的傳播和閱讀
新增 可以針對一些特殊版塊，控制用戶是否需要使用積分購買才可進入
新增 可以批量修改用戶的用戶組
新增 可以對帖子查看頁面用戶頭像下面的參數進行自由調整
新增 可以對指定的版塊、群組，進行主題直播推廣，讓熱門活動更具及時性和參與性
新增 瀏覽帖子的時候，右下角增加快速跟帖、快速返回列表、快速置頂的操作菜單
新增 使用免費的道具，不再需要購買後才可使用。諸如如果設置塗鴉板為免費道具，無需折騰，直接就可以使用
新增 發佈主題的時候，可以使用目錄浮動引導功能。讓具有多章節的主題（比如小說站）更加適合閱讀
新增 發佈主題的時候，可以對帖子內容進行加密，讓知道密碼的用戶才可以查看
新增 用戶的名片，按照性別使用不同的顏色顯示，男女不同更明顯
新增 顯示用戶上次登錄的IP、時間提醒，讓帳號更加安全
新增 查看主題的時候，可以查看熱門回帖、可以屏蔽指定人的回帖、可以屏蔽水帖，讓看帖體驗更加舒暢
新增 查看主題的時候，可以選擇「精簡」模式，並可自由切換。精簡模式下，內容為王
新增 發起投票的時候，可以同時上傳圖片，從而針對圖片進行投票
新增 版塊列表、看帖頁面的用戶名的顏色，根據用戶組設定，進行不同的展示，用戶身份更加突出
新增 查看主題的圖片的時候，可以將好的圖片收藏到自己的相冊裡面
新增 版塊列表自動加載下一頁，同時會剔除可能重複的主題，讓翻頁操作成為過去式
新增 引導遊客註冊，主題裡面的精彩圖片，默認只顯示縮略圖，需要成為會員才可以查看大圖
新增 發佈主題的時候，可以將主題內容自動成為圖片（用戶組權限控制）
新增 增加圖酷功能，把帖子中的圖片集中到一起查看，讀圖時代到來
新增 有新提醒的提示改進，提醒分類，更加便於查看提醒
